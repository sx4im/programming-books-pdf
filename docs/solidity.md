# Solidity

Contract-oriented language for Ethereum and EVM smart contracts.

This page lists curated books and learning resources for this language, organized from beginner to advanced. Most books are shared via public Google Drive links maintained by the project. See [Contributing](../CONTRIBUTING.md) before suggesting new links.

## For Beginners

These books introduce blockchain basics and writing your first smart contracts.

- [**Mastering Ethereum**](https://github.com/ethereumbook/ethereumbook) (*1st Edition*)
- [**Solidity by Example**](https://solidity-by-example.org/) (*Online Resource*)
- [**CryptoZombies**](https://cryptozombies.io/) (*Online Resource*)

## For Intermediate Developers

Move beyond tutorials to build secure, production-minded contracts.

- [**Mastering Blockchain Programming with Solidity**](https://www.packtpub.com/product/mastering-blockchain-programming-with-solidity/9781839218262) (*1st Edition*)
- [**Hands-On Smart Contract Development with Solidity and Ethereum**](https://www.oreilly.com/library/view/hands-on-smart-contract/9781492045250/) (*1st Edition*)

## For Advanced Programmers

Deep dives into security, gas optimization, and protocol design.

- [**Ethereum Smart Contract Security Best Practices**](https://consensys.github.io/smart-contract-best-practices/) (*Online Resource*)
- [**Ethernaut / Capture the Ether style challenges**](https://ethernaut.openzeppelin.com/) (*Online Resource*)

## For Specialized Fields

Focus on DeFi, NFTs, and full-stack dApp development.

- [**Building Ethereum DApps**](https://www.manning.com/books/building-ethereum-dapps) (*1st Edition*)
- [**Ethereum Developer Documentation**](https://ethereum.org/developers/docs/) (*Online Resource*)
