# Objective-C

Object-oriented language for classic macOS and iOS Cocoa development.

This page lists curated books and learning resources for this language, organized from beginner to advanced. Most books are shared via public Google Drive links maintained by the project. See [Contributing](../CONTRIBUTING.md) before suggesting new links.

## For Beginners

These books introduce Objective-C syntax and Cocoa fundamentals.

- [**Programming in Objective-C**](https://www.amazon.com/Programming-Objective-C-6th-Developers-Library/dp/0321967607) (*6th Edition*)
- [**Objective-C Programming: The Big Nerd Ranch Guide**](https://www.amazon.com/Objective-C-Programming-Nerd-Ranch-Guides/dp/032194206X) (*2nd Edition*)
- [**Learn Objective-C on the Mac**](https://www.amazon.com/Learn-Objective-C-Mac-iOS/dp/1430241888) (*2nd Edition*)

## For Intermediate Developers

Once you know the basics, these books help you build real Cocoa apps.

- [**Cocoa Programming for OS X**](https://www.amazon.com/Cocoa-Programming-OS-Nerd-Ranch/dp/0134076958) (*5th Edition*)
- [**iOS Programming: The Big Nerd Ranch Guide**](https://www.amazon.com/iOS-Programming-Nerd-Ranch-Guide/dp/0135264022) (*7th Edition*)

## For Advanced Programmers

Deep dives into runtime, memory management, and bridging to Swift.

- [**Effective Objective-C 2.0**](https://www.amazon.com/Effective-Objective-C-2-0-Specific/dp/0321917014) (*1st Edition*)
- [**Advanced Mac OS X Programming**](https://www.amazon.com/Advanced-Mac-OS-Programming-Developers/dp/0321706583) (*Latest Edition*)

## For Specialized Fields

Focus on legacy maintenance, interoperability, and Apple frameworks.

- [**Learning Cocoa with Objective-C**](https://www.oreilly.com/library/view/learning-cocoa-with/9781491901397/) (*4th Edition*)
- [**Objective-C for Absolute Beginners (iOS path)**](https://www.amazon.com/Objective-C-Absolute-Beginners-iPhone-Programming/dp/1484247041) (*5th Edition*)
