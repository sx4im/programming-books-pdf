# PowerShell

Task automation and configuration management for Windows and cross-platform DevOps.

This page lists curated books and learning resources for this language, organized from beginner to advanced. Most books are shared via public Google Drive links maintained by the project. See [Contributing](../CONTRIBUTING.md) before suggesting new links.

## For Beginners

These books introduce PowerShell syntax, the pipeline, and everyday automation.

- [**Learn Windows PowerShell in a Month of Lunches**](https://www.manning.com/books/learn-powershell-in-a-month-of-lunches-fourth-edition) (*4th Edition*)
- [**PowerShell for Sysadmins**](https://nostarch.com/powershellsysadmins) (*1st Edition*)
- [**Windows PowerShell Cookbook**](https://www.oreilly.com/library/view/windows-powershell-cookbook/9781098164317/) (*4th Edition*)

## For Intermediate Developers

Move beyond one-liners to write robust scripts and modules.

- [**PowerShell in Depth**](https://www.manning.com/books/powershell-in-depth-second-edition) (*2nd Edition*)
- [**Learn PowerShell Scripting in a Month of Lunches**](https://www.manning.com/books/learn-powershell-scripting-in-a-month-of-lunches-second-edition) (*2nd Edition*)

## For Advanced Programmers

Deep dives into remoting, DSC, and production-grade tooling.

- [**The PowerShell Scripting and Toolmaking Book**](https://leanpub.com/powershell-scripting-toolmaking) (*Latest Edition*)
- [**PowerShell for Pentesting**](https://www.amazon.com/PowerShell-Penetration-Testers-Mike-Fletcher/dp/1484293614) (*1st Edition*)

## For Specialized Fields

Focus on Azure, Microsoft 365, and enterprise automation.

- [**Learn Azure in a Month of Lunches**](https://www.manning.com/books/learn-azure-in-a-month-of-lunches-second-edition) (*2nd Edition*)
- [**Azure PowerShell Quick Start Guide**](https://www.packtpub.com/product/azure-powershell-quick-start-guide/9781789614954) (*1st Edition*)
