# Julia

High-performance language for numerical computing, science, and machine learning.

This page lists curated books and learning resources for this language, organized from beginner to advanced. Most books are shared via public Google Drive links maintained by the project. See [Contributing](../CONTRIBUTING.md) before suggesting new links.

## For Beginners

These books introduce Julia's syntax and its appeal for high-performance scientific and numerical computing.

- [**Think Julia: How to Think Like a Computer Scientist**](https://benlauwens.github.io/ThinkJulia.jl/latest/book.html) (*1st Edition*)
- [**Introducing Julia**](https://en.wikibooks.org/wiki/Introducing_Julia) (*Online Resource*)
- [**Julia Programming for Operations Research**](https://www.amazon.com/Julia-Programming-Operations-Research-Optimization/dp/3031464217) (*2nd Edition*)
- [**Julia 1.0 By Example**](https://www.packtpub.com/product/julia-10-by-example/9781788299522) (*1st Edition*)

## For Intermediate Developers

Once you know the basics, these books help you write efficient, idiomatic Julia for science and engineering.

- [**Julia High Performance**](https://www.packtpub.com/product/julia-high-performance-second-edition/9781789349414) (*2nd Edition*)
- [**Learning Scientific Programming with Julia**](https://www.amazon.com/Learning-Scientific-Programming-Julia-Computational/dp/1108445070) (*1st Edition*)
- [**Julia Data Science**](https://github.com/JuliaDataScience/JuliaDataScience) (*Free online book*)

## For Advanced Programmers

Dive deep into Julia's type system, multiple dispatch, metaprogramming, and performance tuning.

- [**Algorithms for Optimization**](https://algorithmsbook.com/optimization/) (*1st Edition*)
- [**Algorithms for Decision Making**](https://algorithmsbook.com/) (*1st Edition*)
- [**Julia 1.7 Cookbook**](https://www.packtpub.com/product/julia-17-cookbook/9781803237587) (*1st Edition*)

## For Specialized Fields

Focus on applying Julia to data analysis, statistics, and machine learning.

- [**Statistics with Julia**](https://www.amazon.com/Statistics-Julia-Fundamentals-Data-Science/dp/3030401507) (*1st Edition*)
- [**Machine Learning with Julia**](https://www.packtpub.com/product/machine-learning-with-julia/9781839218408) (*1st Edition*)
- [**Quantitative Economics with Julia**](https://julia.quantecon.org/intro.html) (*Online Resource*)
