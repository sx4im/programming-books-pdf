# Elixir

Functional language on the Erlang VM for concurrent, fault-tolerant systems.

This page lists curated books and learning resources for this language, organized from beginner to advanced. Most books are shared via public Google Drive links maintained by the project. See [Contributing](../CONTRIBUTING.md) before suggesting new links.

## For Beginners

These books introduce Elixir's functional style and the actor-based concurrency model that runs on the battle-tested Erlang VM.

- [**Elixir Getting Started Guide**](https://elixir-lang.org/getting-started/introduction.html) (*Official free guide*)
- [**Elixir School**](https://elixirschool.com/) (*Online Resource*)
- [**Programming Elixir 1.6**](https://www.amazon.com/Programming-Elixir-1-6-Functional-Concurrent/dp/1680502999) (*1.6 Edition*)
- [**Learn Functional Programming with Elixir**](https://www.amazon.com/Learn-Functional-Programming-Elixir-Concurrent/dp/1680504568) (*1st Edition*)
- [**Introducing Elixir**](https://www.oreilly.com/library/view/introducing-elixir-2nd/9781492026188/) (*2nd Edition*)

## For Intermediate Developers

Once you know the basics, these books help you build real applications with OTP, processes, and tooling.

- [**Elixir in Action**](https://www.manning.com/books/elixir-in-action-second-edition) (*2nd Edition*)
- [**Craft GraphQL APIs in Elixir with Absinthe**](https://www.manning.com/books/craft-graphql-apis-in-elixir-with-absinthe) (*1st Edition*)
- [**Adopting Elixir**](https://www.amazon.com/Adopting-Elixir-Concepts-Production-Applications/dp/1680506129) (*1st Edition*)

## For Advanced Programmers

Deep dives into metaprogramming, OTP design, and production-grade Elixir systems.

- [**Metaprogramming Elixir**](https://www.amazon.com/Metaprogramming-Elixir-Write-Less-Code/dp/1680500414) (*1st Edition*)
- [**Designing Elixir Systems with OTP**](https://www.amazon.com/Designing-Elixir-Systems-OTP/dp/1680506617) (*1st Edition*)
- [**Property-Based Testing with PropEr, Erlang, and Elixir**](https://www.manning.com/books/property-based-testing-with-proper-erlang-and-elixir) (*1st Edition*)

## For Specialized Fields

Focus on Phoenix, Ecto, LiveView, and real-time web systems.

- [**Programming Phoenix 1.4**](https://www.amazon.com/Programming-Phoenix-1-4-Productive-Reliable/dp/1680502265) (*1.4 Edition*)
- [**Programming Ecto**](https://www.amazon.com/Programming-Ecto-Build-Database-Applications/dp/1680507380) (*1st Edition*)
- [**Programming Phoenix LiveView**](https://www.amazon.com/Programming-Phoenix-LiveView-Interactive-Applications/dp/1680508212) (*1st Edition*)
