# VBA

Automation language embedded in Microsoft Office for macros and business workflows.

This page lists curated books and learning resources for this language, organized from beginner to advanced. Most books are shared via public Google Drive links maintained by the project. See [Contributing](../CONTRIBUTING.md) before suggesting new links.

## For Beginners

These books are perfect if you are new to VBA or Office automation.

- [**Excel VBA Programming For Dummies**](https://www.amazon.com/Excel-VBA-Programming-Dummies-Computer/dp/1119849176) (*6th Edition*)
- [**VBA For Dummies**](https://www.amazon.com/VBA-Dummies-John-Paul-Mueller/dp/1119247260) (*5th Edition*)
- [**Microsoft Excel 365 Bible**](https://www.amazon.com/Microsoft-Excel-365-Bible-Mastering/dp/1119835108) (*Latest Edition*)

## For Intermediate Developers

Once you know the basics, these books help you build robust Office macros and add-ins.

- [**Excel 2019 Power Programming with VBA**](https://www.amazon.com/Excel-2019-Power-Programming-VBA/dp/1119514924) (*1st Edition*)
- [**Mastering VBA for Microsoft Office 365**](https://www.amazon.com/Mastering-Microsoft-Office-Richard-Mansfield/dp/1119578841) (*Latest Edition*)

## For Advanced Programmers

Deep dives into class modules, APIs, and professional VBA architecture.

- [**Professional Excel Development**](https://www.amazon.com/Professional-Excel-Development-Definitive-Applications/dp/0321508794) (*2nd Edition*)
- [**Advanced Excel Essentials**](https://www.amazon.com/Advanced-Excel-Essentials-Jordan-Goldmeier/dp/1484207343) (*1st Edition*)

## For Specialized Fields

Focus on finance models, Access databases, and Outlook automation.

- [**Access 2019 Bible**](https://www.amazon.com/Access-2019-Bible-Michael-Alexander/dp/1119515629) (*1st Edition*)
- [**Financial Modeling in Excel / VBA**](https://www.amazon.com/Financial-Modeling-Simon-Benninga/dp/0262027283) (*4th Edition*)
