# Perl

Flexible language renowned for text processing, sysadmin, and bioinformatics scripts.

This page lists curated books and learning resources for this language, organized from beginner to advanced. Most books are shared via public Google Drive links maintained by the project. See [Contributing](../CONTRIBUTING.md) before suggesting new links.

## For Beginners

These books are perfect if you are new to Perl or coming from another scripting language.

- [**Learning Perl**](https://www.oreilly.com/library/view/learning-perl-8th/9781492094920/) (*8th Edition*)
- [**Intermediate Perl**](https://www.oreilly.com/library/view/intermediate-perl-2nd/9781449343781/) (*2nd Edition*)
- [**Beginning Perl**](https://www.amazon.com/Beginning-Perl-Curtis-Poe/dp/1118013840) (*1st Edition*)

## For Intermediate Developers

Once you know the basics, these books help you write idiomatic and maintainable Perl.

- [**Modern Perl**](https://www.modernperlbooks.com/books/modern_perl_2014/) (*2014 Edition*)
- [**Programming Perl (the Camel Book)**](https://www.oreilly.com/library/view/programming-perl-4th/9781449321451/) (*4th Edition*)

## For Advanced Programmers

Deep dives into advanced data structures, OO, and performance.

- [**Higher-Order Perl**](https://hop.perl.plover.com/) (*1st Edition*)
- [**Mastering Perl**](https://www.oreilly.com/library/view/mastering-perl-2nd/9781449365233/) (*2nd Edition*)

## For Specialized Fields

Focus on bioinformatics, network programming, and legacy modernization.

- [**Beginning Perl for Bioinformatics**](https://www.oreilly.com/library/view/beginning-perl-for/0596000804/) (*1st Edition*)
- [**Network Programming with Perl**](https://www.amazon.com/Network-Programming-Perl-Lincoln-Stein/dp/0201615711) (*1st Edition*)
