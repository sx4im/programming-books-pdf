# Zig

Modern systems language focused on simplicity, safety, and explicit control.

This page lists curated books and learning resources for this language, organized from beginner to advanced. Most books are shared via public Google Drive links maintained by the project. See [Contributing](../CONTRIBUTING.md) before suggesting new links.

## For Beginners

These resources introduce Zig's syntax, tooling, and systems-programming mindset.

- [**Zig Language Reference**](https://ziglang.org/documentation/master/) (*Online Resource*)
- [**Introduction to Zig**](https://pedropark99.github.io/zig-book/) (*Free online book*)
- [**Ziglearn**](https://ziglearn.org/) (*Online Resource*)

## For Intermediate Developers

Build practical skills with allocators, build system, and C interop.

- [**Zig Guide**](https://zig.guide/) (*Online Resource*)
- [**Learning Zig**](https://www.openmymind.net/learning_zig/) (*Online Resource*)

## For Advanced Programmers

Deep dives into comptime, unsafe patterns done safely, and systems design.

- [**Zig Documentation — Advanced Language Features**](https://ziglang.org/documentation/master/#Language-Reference) (*Online Resource*)
- [**Writing an OS in Zig (community guides)**](https://github.com/ziglang/zig) (*Online Resource*)

## For Specialized Fields

Focus on embedded, cross-compilation, and replacing C toolchains.

- [**Zig Build System Documentation**](https://ziglang.org/learn/build-system/) (*Online Resource*)
- [**Using Zig as a C Compiler / Cross Linker**](https://ziglang.org/learn/overview/) (*Online Resource*)
