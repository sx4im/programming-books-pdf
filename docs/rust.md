# Rust

Memory-safe systems language for performance-critical and reliable software.

This page lists curated books and learning resources for this language, organized from beginner to advanced. Most books are shared via public Google Drive links maintained by the project. See [Contributing](../CONTRIBUTING.md) before suggesting new links.

## For Beginners

These books are for newcomers to Rust and provide a solid foundation in its unique concepts of ownership and borrowing.

- [**The Rust Programming Language**](https://drive.google.com/file/d/1EMwXxNDPLMugxczR1KsSakNTxCDlxFgm/view?usp=drive_link) (*2nd Edition*)
- [**Programming Rust**](https://drive.google.com/file/d/1YPjM92aviYus1ZngUdvW72dLpxtbYIVv/view?usp=drive_link) (*2nd Edition*)
- [**Rust by Example**](https://doc.rust-lang.org/rust-by-example/) (*Online Resource*)
- [**Beginning Rust**](https://drive.google.com/file/d/15Smb7lVBLcjdAUDY0oYbV7xuvRPq22E3/view?usp=drive_link) (*2nd Edition*)

## For Intermediate Developers

These books will help you level up your skills and learn to write truly idiomatic and effective Rust code.

- [**Rust for Rustaceans**](https://drive.google.com/file/d/1vCglEaaVMujT0Z2_fjry5evTTYcNLEyj/view?usp=drive_link) (*1st Edition*)
- [**Rust in Action**](https://drive.google.com/file/d/17cQzEkMvBFKSIbYYGFizYoQ0S1bZ8l4F/view?usp=drive_link) (*1st Edition*)
- [**Command-Line Rust**](https://drive.google.com/file/d/1f4ZOLaIQ8pNTA5gT5bXOXxZhZP5ZDhiL/view?usp=drive_link) (*1st Edition*)
- [**Mastering Rust**](https://drive.google.com/file/d/18rol5LmfLALJPSTLG-lLI3FlGqkigEBA/view?usp=drive_link) (*2nd Edition*)

## For Advanced Programmers

Dive deep into Rust's most powerful and complex features, including concurrency, performance, and unsafe code.

- [**The Rustonomicon**](https://doc.rust-lang.org/nomicon/) (*Online Resource*)
- [**Programming WebAssembly with Rust**](https://drive.google.com/file/d/1fA-QBP7oMWIa-JliX6lnguT-kZPbgJwX/view?usp=drive_link) (*1st Edition*)
- [**Hands-On Concurrency with Rust**](https://drive.google.com/file/d/108VatywkKNA59u__xdIXgfcELeFPGHYn/view?usp=drive_link) (*1st Edition*)
- [**Rust High Performance**](https://drive.google.com/file/d/1PRqqydj7tGarWkFgEQMYN5fKS3G6QD16/view?usp=drive_link) (*2nd Edition*)
- [**Black Hat Rust**](https://drive.google.com/file/d/1aiT-Luw2WcVkps7BaDYgO7Web1idqvX_/view?usp=drive_link) (*1st Edition*)

## For Specialized Fields

Focus on applying Rust to a specific professional domain, such as web services, embedded systems, or GUI applications.

- [**Zero To Production In Rust**](https://drive.google.com/file/d/1xOHEXyd3dF3e2MlfRVBSQ2P09gsr-ZIt/view?usp=drive_link) (*1st Edition*)
- [**The Embedded Rust Book**](https://drive.google.com/file/d/1nskPjDJBUesrfDuWp49In7DfFTFxDxoX/view?usp=sharing) (*Online Resource*)
- [**Rust Web development**](https://drive.google.com/file/d/1HuYUELERFfoNpuyAf6oWxzd3WX_MkSNa/view?usp=sharing) (*2nd Edition*)
- [**Game Development with Rust**](https://drive.google.com/file/d/1ymZXvXOycCZJDJMLURXKSdc4QF8CbPFX/view?usp=sharing) (*1st Edition*)
- [**Hands-On Data Structures and Algorithms with Rust**](https://drive.google.com/file/d/15bDvvz9Vu4Wo6tkfkai1bR37AYLW3XC5/view?usp=sharing) (*2nd Edition*)
