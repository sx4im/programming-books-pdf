# Groovy

Dynamic JVM language used in Gradle, Jenkins pipelines, and scripting.

This page lists curated books and learning resources for this language, organized from beginner to advanced. Most books are shared via public Google Drive links maintained by the project. See [Contributing](../CONTRIBUTING.md) before suggesting new links.

## For Beginners

These books introduce Groovy's concise syntax for Java developers and newcomers.

- [**Groovy in Action**](https://www.manning.com/books/groovy-in-action-second-edition) (*2nd Edition*)
- [**Making Java Groovy**](https://www.manning.com/books/making-java-groovy) (*1st Edition*)
- [**Groovy Programming Fundamentals**](https://groovy-lang.org/documentation.html) (*Online Resource*)

## For Intermediate Developers

Once you know the basics, these books help you use Groovy for DSLs and build tooling.

- [**Gradle in Action**](https://www.manning.com/books/gradle-in-action) (*1st Edition*)
- [**Programming Groovy 2**](https://pragprog.com/titles/vslg2/programming-groovy-2/) (*2nd Edition*)

## For Advanced Programmers

Deep dives into metaprogramming, AST transforms, and JVM integration.

- [**Groovy 2 Cookbook**](https://www.packtpub.com/product/groovy-2-cookbook/9781849519366) (*1st Edition*)
- [**Mastering Groovy**](https://www.packtpub.com/product/mastering-groovy/9781786462336) (*1st Edition*)

## For Specialized Fields

Focus on Jenkins pipelines, Grails, and build automation.

- [**Jenkins 2: Up and Running**](https://www.oreilly.com/library/view/jenkins-2-up/9781491979587/) (*1st Edition*)
- [**Grails 3 — Guide / docs**](https://docs.grails.org/) (*Online Resource*)
