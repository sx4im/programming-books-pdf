# Dart

Language behind Flutter for cross-platform mobile, web, and desktop apps.

This page lists curated books and learning resources for this language, organized from beginner to advanced. Most books are shared via public Google Drive links maintained by the project. See [Contributing](../CONTRIBUTING.md) before suggesting new links.

## For Beginners

These books introduce Dart's syntax and the basics of building cross-platform apps, often through the Flutter framework.

- [**Flutter for Dummies**](https://drive.google.com/file/d/1F9Ep5ATcQPC9f2BO7dE6HtZ2isQZcX79/view?usp=sharing) (*1st Edition*)
- [**Dart Apprentice: Fundamentals**](https://drive.google.com/file/d/1uEsVKWLocoPIeIesjS_4lLIJiIsOJE2X/view?usp=sharing) (*1st Edition*)
- [**Flutter Apprentice**](https://drive.google.com/file/d/193ZpR1quIK79wFDwhiZw4W4UqckYVt0W/view?usp=sharing) (*4th Edition*)
- [**Dart: Up and Running**](https://drive.google.com/file/d/1gareIc2dTaKFBuT_EClXC-M75uhn7TLE/view?usp=sharing) (*1st Edition*)
- [**Beginning App Development with Flutter**](https://drive.google.com/file/d/1hescnMo5wjJhOL9U8KESqAVJyPmv7Ipz/view?usp=sharing) (*2nd Edition*)

## For Intermediate Developers

Move beyond the basics to build robust, well-structured applications with Dart and Flutter.

- [**Programming Flutter**](https://drive.google.com/file/d/113W5J6-MhLT0JV0Y2MNv0UgN8cKTtmJv/view?usp=sharing) (*2020 Edition*)
- [**Flutter in Action**](https://drive.google.com/file/d/1SmiglCBKIGbLCuhLXitfApG6TG2Chxbw/view?usp=sharing) (*1st Edition*)
- [**Dart Apprentice: Beyond the Basics**](https://drive.google.com/file/d/1m_TcVG6jj2ZUnwuo_b2TyRboNEIFrJjl/view?usp=sharing) (*1st Edition*)
- [**Pragmatic Flutter**](https://drive.google.com/file/d/1t7d8WFr1UiDWBsWQhMM43XdDXb6ojJsN/view?usp=sharing) (*1st Edition*)

## For Advanced Programmers

Deep dives into application architecture, state management, and high-performance Dart.

- [**Real-World Flutter by Tutorials**](https://drive.google.com/file/d/16ynNoqtHexXw6JlCrXHDZEQ7W9UIQA6Y/view?usp=sharing) (*1st Edition*)
- [**Mastering Dart**](https://drive.google.com/file/d/1YDiXxXdbIth7olLR7oAX8QVgER2FdBeM/view?usp=sharing) (*1st Edition*)
