# Scala

JVM language combining object-oriented and functional programming for big data and backends.

This page lists curated books and learning resources for this language, organized from beginner to advanced. Most books are shared via public Google Drive links maintained by the project. See [Contributing](../CONTRIBUTING.md) before suggesting new links.

## For Beginners

These books provide a solid foundation in Scala's hybrid object-oriented and functional style, ideal for newcomers and Java developers alike.

- [**Scala 3 Book**](https://docs.scala-lang.org/scala3/book/introduction.html) (*Official free online book*)
- [**Creative Scala**](https://www.creativescala.org/) (*Free online introduction*)
- [**Essential Scala**](https://underscore.io/books/essential-scala/) (*Online Resource*)
- [**Programming in Scala**](https://www.manning.com/books/programming-in-scala-fifth-edition) (*5th Edition*)
- [**Scala for the Impatient**](https://www.oreilly.com/library/view/scala-for-the-impatient/9780134540566/) (*3rd Edition*)

## For Intermediate Developers

Move beyond the basics to write idiomatic, effective Scala and master its rich standard library.

- [**Tour of Scala**](https://docs.scala-lang.org/tour/) (*Official language tour*)
- [**Hands-on Scala Programming**](https://www.handsonscala.com/) (*1st Edition*)
- [**Functional Programming in Scala**](https://www.manning.com/books/functional-programming-in-scala) (*1st Edition*)
- [**Scala Cookbook**](https://www.oreilly.com/library/view/scala-cookbook/9781449319385/) (*1st Edition*)

## For Advanced Programmers

Dive deep into functional programming theory, type-level programming, and the most powerful features of the language.

- [**Scala with Cats**](https://www.scalawithcats.com/) (*Online Resource*)
- [**Zionomicon**](https://www.zionomicon.com/) (*Online Resource*)
- [**Functional Programming for Mortals with Scalaz**](https://leanpub.com/fpmortals) (*Latest Edition*)

## For Specialized Fields

Focus on applying Scala to data engineering, distributed systems, and analytics.

- [**Spark: The Definitive Guide**](https://www.oreilly.com/library/view/spark-the-definitive/9781491912218/) (*1st Edition*)
- [**Scala and Spark for Big Data Analytics**](https://www.packtpub.com/product/scala-and-spark-for-big-data-analytics/9781783550500) (*1st Edition*)
- [**Akka in Action**](https://www.manning.com/books/akka-in-action) (*2nd Edition*)
