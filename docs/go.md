# Go

Simple, concurrent language from Google for cloud services, CLI tools, and infrastructure.

This page lists curated books and learning resources for this language, organized from beginner to advanced. Most books are shared via public Google Drive links maintained by the project. See [Contributing](../CONTRIBUTING.md) before suggesting new links.

## For Beginners

These books are perfect for newcomers to Go, whether you are a new programmer or coming from another language.

- [**The Go Programming Language**](https://drive.google.com/file/d/1u4q3ykHUxlwxrDBN9tyCoZaSzjJ7_8RN/view?usp=sharing) (*1st Edition*)
- [**Learning Go**](https://drive.google.com/file/d/1QenR-5Oj8sqk0d89enT9wZZsleLu13Df/view?usp=drive_link) (*2nd Edition*)
- [**Head First Go**](https://drive.google.com/file/d/1xWd4eyqz_7T341W77naEltuDw6rgd3mW/view?usp=drive_link) (*1st Edition*)
- [**Get Programming with Go**](https://drive.google.com/file/d/1BTvPt7YDQF8XgR-YtU8S5cFTppsWVW0_/view?usp=drive_link) (*1st Edition*)

## For Intermediate Developers

These books will help you master Go's most powerful features and understand how to build professional applications.

- [**Go in Action**](https://drive.google.com/file/d/1J3ZfepWnOw39BpjWlnOXHU7MPFbPd-zi/view?usp=drive_link) (*1st Edition*)
- [**Concurrency in Go**](https://drive.google.com/file/d/1rdB_l3_Jm-SPHenMKF0KLCTd8SQ4GUwj/view?usp=drive_link) (*1st Edition*)
- [**Mastering Go**](https://drive.google.com/file/d/1J4QcnUrF7LVIeeGtQlUGM5wXrgmEcDC3/view?usp=drive_link) (*4th Edition*)
- [**Go Programming Cookbook**](https://drive.google.com/file/d/1HFnAUXW-_NkIou9evJUh3HX7JAt_t4He/view?usp=drive_link) (*2nd Edition*)
- [**Go in Practice**](https://drive.google.com/file/d/1dwQ8BZIHzAJsOpmCowBRNhFDju1RFh8O/view?usp=drive_link) (*2nd Edition*)

## For Advanced Programmers

Dive deep into systems design, cloud-native development, and advanced Go features.

- [**Designing Data-Intensive Applications**](https://drive.google.com/file/d/19G8U2Yx1XZOnyl-0tcPrSL2yjUXake7J/view?usp=drive_link) (*1st Edition*)
- [**Cloud Native Go**](https://drive.google.com/file/d/171ZeP-RMQRDVT8mdBAzZiNpnB2mXfkzu/view?usp=drive_link) (*2nd Edition*)
- [**Network Programming with Go**](https://drive.google.com/file/d/1J6FcdLAhVvdGA08PmEKP1jyVjrAchhry/view?usp=drive_link) (*1st Edition*)
- [**100 Go Mistakes and How to Avoid Them**](https://drive.google.com/file/d/1pZCiQxWCQJN7qRfoHmBREQBtwzpesg1R/view?usp=drive_link) (*1st Edition*)

## For Specialized Fields

Focus on applying Go to a specific professional domain, such as web development or security.

- [**Let's Go**](https://drive.google.com/file/d/1nwvxTD7SxHUCKKSBJn2HEwsg0f8R7pwP/view?usp=drive_link) (*2nd Edition*)
- [**Let's Go Further**](https://drive.google.com/file/d/1eBwXCyKKOnN94vawC-GXtW5iZNs6jT3l/view?usp=drive_link) (*1st Edition*)
- [**Black Hat Go**](https://drive.google.com/file/d/19MikaztAUtBXsn-oCEoxPzO9szsNguqk/view?usp=drive_link) (*1st Edition*)
- [**Writing An Interpreter In Go**](https://drive.google.com/file/d/1pnoYA-LvaXZWAPkGpgdO6gnfwmSSv-F0/view?usp=drive_link) (*1st Edition*)
