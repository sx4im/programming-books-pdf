# Clojure

Modern Lisp on the JVM for concurrency, data, and functional systems.

This page lists curated books and learning resources for this language, organized from beginner to advanced. Most books are shared via public Google Drive links maintained by the project. See [Contributing](../CONTRIBUTING.md) before suggesting new links.

## For Beginners

These books introduce Clojure's Lisp heritage and immutable data model.

- [**Clojure for the Brave and True**](https://www.braveclojure.com/clojure-for-the-brave-and-true/) (*1st Edition*)
- [**Clojure in Action**](https://www.manning.com/books/clojure-in-action) (*2nd Edition*)
- [**Living Clojure**](https://www.oreilly.com/library/view/living-clojure/9781491909270/) (*1st Edition*)

## For Intermediate Developers

Move beyond the basics to write idiomatic Clojure and use the REPL effectively.

- [**Clojure Programming**](https://www.oreilly.com/library/view/clojure-programming/9781449310387/) (*1st Edition*)
- [**The Joy of Clojure**](https://www.manning.com/books/the-joy-of-clojure-second-edition) (*2nd Edition*)

## For Advanced Programmers

Deep dives into macros, transducers, and concurrent design.

- [**Clojure: The Essential Reference**](https://www.manning.com/books/clojure-the-essential-reference) (*1st Edition*)
- [**Professional Clojure**](https://www.amazon.com/Professional-Clojure-Jeremy-Anderson/dp/1119267277) (*1st Edition*)

## For Specialized Fields

Focus on web apps, data pipelines, and ClojureScript.

- [**Web Development with Clojure**](https://pragprog.com/titles/dswdcloj3/web-development-with-clojure-third-edition/) (*3rd Edition*)
- [**ClojureScript Unraveled**](https://www.amazon.com/ClojureScript-Unraveled-Andrey-Antov/dp/1680503006) (*Latest Edition*)
