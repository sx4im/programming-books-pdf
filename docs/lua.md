# Lua

Lightweight scripting language popular in games, Neovim, and embedded systems.

This page lists curated books and learning resources for this language, organized from beginner to advanced. Most books are shared via public Google Drive links maintained by the project. See [Contributing](../CONTRIBUTING.md) before suggesting new links.

## For Beginners

These books introduce Lua's simple syntax and its role as an embeddable scripting language.

- [**Programming in Lua**](https://www.lua.org/pil/) (*4th Edition*)
- [**Beginning Lua Programming**](https://www.amazon.com/Beginning-Lua-Programming-Kurt-Jung/dp/0470069171) (*1st Edition*)
- [**Lua Quick Start Guide**](https://www.packtpub.com/product/lua-quick-start-guide/9781789343229) (*1st Edition*)

## For Intermediate Developers

Move beyond the basics to write idiomatic Lua and integrate it with host applications.

- [**Lua Programming Gems**](https://www.lua.org/gems/) (*1st Edition*)
- [**Creating Solid APIs with Lua**](https://www.lua.org/docs.html) (*Online Resource*)

## For Advanced Programmers

Deep dives into the Lua C API, performance, and language internals.

- [**Lua 5.4 Reference Manual**](https://www.lua.org/manual/5.4/) (*Official Reference*)
- [**Mastering Lua Programming**](https://www.packtpub.com/product/mastering-lua-programming/9781785284359) (*1st Edition*)

## For Specialized Fields

Focus on game scripting, Roblox, and Neovim plugin development.

- [**Roblox Lua Game Development**](https://www.amazon.com/Coding-Roblox-Games-Made-Easy/dp/1803234678) (*2nd Edition*)
- [**Crafting Interpreters (Lua-friendly patterns)**](https://craftinginterpreters.com/) (*1st Edition*)

## Comprehensive References

These resources are perfect for quick lookups and staying current with the language.

- [**Official Lua Documentation**](https://www.lua.org/docs.html) (*Online Resource*)
- [**Lua Users Wiki**](https://lua-users.org/wiki/) (*Online Resource*)
