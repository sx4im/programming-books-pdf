Markup and styling languages that power the modern web.

This page lists curated books and learning resources for this language, organized from beginner to advanced. Most books are shared via public Google Drive links maintained by the project. See [Contributing](../CONTRIBUTING.md) before suggesting new links.

## For Beginners

These books are perfect if you are new to building web pages with HTML and CSS.

- [**HTML and CSS: Design and Build Websites**](https://www.amazon.com/HTML-CSS-Design-Build-Websites/dp/1118008189) (*1st Edition*)
- [**Learning Web Design**](https://www.oreilly.com/library/view/learning-web-design/9781491960196/) (*5th Edition*)
- [**HTML5 for Web Designers**](https://www.amazon.com/HTML5-Web-Designers-Jeremy-Keith/dp/1937557208) (*2nd Edition*)
- [**CSS: The Definitive Guide**](https://www.oreilly.com/library/view/css-the-definitive/9781098117603/) (*5th Edition*)

## For Intermediate Developers

Once you know the basics, these books help you write modern, maintainable layouts and UI patterns.

- [**CSS in Depth**](https://www.manning.com/books/css-in-depth) (*1st Edition*)
- [**Every Layout**](https://every-layout.dev/) (*Online Resource*)
- [**Responsive Web Design**](https://www.amazon.com/Responsive-Web-Design-Ethan-Marcotte/dp/1937557186) (*2nd Edition*)
- [**CSS Secrets**](https://www.oreilly.com/library/view/css-secrets/9781449372736/) (*1st Edition*)

## For Advanced Programmers

Deep dives into architecture, performance, and advanced CSS techniques.

- [**CSS: The Missing Manual**](https://www.oreilly.com/library/view/css-the-missing/9781491985038/) (*4th Edition*)
- [**Transcending CSS**](https://www.amazon.com/Transcending-CSS-Value-Aesthetic-Design/dp/0321412416) (*1st Edition*)

## For Specialized Fields

Focus on accessibility, design systems, and production front-end craft.

- [**Inclusive Design Patterns**](https://www.smashingmagazine.com/printed-books/inclusive-design-patterns/) (*1st Edition*)
- [**Designing Interfaces**](https://www.oreilly.com/library/view/designing-interfaces-3rd/9781492051954/) (*3rd Edition*)
