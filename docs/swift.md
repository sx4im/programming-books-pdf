# Swift

Apple’s language for iOS, macOS, and Apple-platform application development.

This page lists curated books and learning resources for this language, organized from beginner to advanced. Most books are shared via public Google Drive links maintained by the project. See [Contributing](../CONTRIBUTING.md) before suggesting new links.

## For Beginners

These books focus on the Swift language syntax and the basics of building your first mobile applications using modern Apple standards.

- [**Swift Apprentice**](https://drive.google.com/file/d/1tA1abtGxZvJUuAIyCA6TJbkzGPKNhuHg/view?usp=sharing) (*7th Edition*)
- [**Head First Swift**](https://drive.google.com/file/d/1aV5_twjheTblaadPX0Up9CHU7XydT2T1/view?usp=sharing) (*1st Edition*)
- [**Swift Programming: The Big Nerd Ranch Guide**](https://drive.google.com/file/d/1b9t6A265KOdy99PTDKrFnRxe_JPmbW-1/view?usp=sharing) (*3rd Edition*)
- [**Develop in Swift Fundamentals**](https://developer.apple.com/learn/curriculum/) (*Official Apple curriculum*)
- [**SwiftUI for Masterminds**](https://drive.google.com/file/d/1IWIUDcvNQbI2zNZ_L2lBoEnwGUKpinLT/view?usp=sharing) (*3rd Edition*)
- [**IOS Development with Swift**](https://drive.google.com/file/d/1rxN7h9etXSJPKlGdq1ooNhHPDqhpIS9U/view?usp=sharing)

## For Intermediate Developers

These books help you move from writing basic code to building robust, professional-grade applications with clean architecture.

- [**Learning Swift**](https://drive.google.com/file/d/1we9nE1pbHEJ9hBSWHNX5tiYObBRyhbjc/view?usp=sharing) (*3rd Edition*)
- [**Swift in Depth**](https://drive.google.com/file/d/1Mi4-0eInG74YugNvLj1i6M8Py60uf7TU/view?usp=sharing) (*1st Edition*)
- [**Swift Gems**](https://drive.google.com/file/d/1ynPRUtdPfh1H8JCTZ8EFFtcN27pG6m4p/view?usp=sharing) (*1st Edition - 2025*)
- [**Practical Swift**](https://drive.google.com/file/d/1b83V83-WyEWEMp5fn_ytmm5njuDvQYE0/view?usp=sharing)

## For Advanced Programmers

Deep dives into the compiler, performance tuning, and the most complex parts of the Swift type system and concurrency model.

- [**Advanced Swift**](https://drive.google.com/file/d/1kT8szQJtNBkAzHDZMTTv5w3u7qHrqgNf/view?usp=sharing) (*Latest Edition*)
- [**Modern Concurrency in Swift**](https://drive.google.com/file/d/16tMu59OYTMKOUxtfc9AvY1RmiCLcc_n3/view?usp=sharing)
- [**Expert Swift**](https://drive.google.com/file/d/1rMtUlu7cd7N9gODzhDikQUwSTUmCSQ2x/view?usp=sharing) (*1st Edition*)
- [**Thinking in SwiftUI**](https://drive.google.com/file/d/12W4CAVOxa4IPuiIqoztRDWhbhf0fiQ02/view?usp=sharing) (*Latest Edition*)

Focused on professional domains like app architecture, test-driven development, and high-performance graphics.
