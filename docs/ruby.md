# Ruby

Expressive language known for Rails and developer-friendly web development.

This page lists curated books and learning resources for this language, organized from beginner to advanced. Most books are shared via public Google Drive links maintained by the project. See [Contributing](../CONTRIBUTING.md) before suggesting new links.

## For Beginners

These books are perfect for newcomers to programming, or developers coming from another language who want to learn Ruby's elegant, expressive syntax.

- [**The Well-Grounded Rubyist**](https://drive.google.com/file/d/1Cf52TXHVNxnmiormIZxnCAIeTStlqMxN/view?usp=sharing) (*3rd Edition*)
- [**Learn to Program**](https://drive.google.com/file/d/1EwtvkYvk4mCYLCnkvuG98LX4Wufpg1mZ/view?usp=sharing) (*3rd Edition*)
- [**Head First Ruby**](https://drive.google.com/file/d/17ad--m9IvySUTM80Tt_1vUvNOlbveDjk/view?usp=sharing) (*1st Edition*)
- [**Beginning Ruby: From Novice to Professional**](https://drive.google.com/file/d/1oQvt0rSY30t8kNP5ns_lEe_tjEZfF68P/view?usp=sharing) (*3rd Edition*)
- [**The Ruby Way**](https://drive.google.com/file/d/1a3ZoDJBGjwAbtoZATusPisqY1nz00E_J/view?usp=sharing) (*3rd Edition*)

## For Intermediate Developers

Once you know the basics, these books will help you write idiomatic, object-oriented, and maintainable Ruby code.

- [**Eloquent Ruby**](https://drive.google.com/file/d/1ZZn9kyzM86xojOtO4Bgj8CRqVWupc5yx/view?usp=sharing) (*1st Edition*)
- [**Practical Object-Oriented Design in Ruby (POODR)**](https://drive.google.com/file/d/1aAlGlsTO_pR427WYvSRraHnKPR6Ydl_X/view?usp=sharing) (*1st Edition*)
- [**Confident Ruby**](https://drive.google.com/file/d/1qtL_lhD2tq2TB6rSpdVf5E7tjmXJYt0J/view?usp=sharing) (*1st Edition*)
- [**Ruby Under a Microscope**](https://drive.google.com/file/d/1mzeyaEwPOtSLWz39vbAxQ90_KyZ1pRpQ/view?usp=sharing) (*1st Edition*)
- [**Programming Ruby 3.3 (The Pickaxe Book)**](https://drive.google.com/file/d/1PlMP53k6bF36OMawAWM1LI31Z4exJ8MG/view?usp=sharing) (*5th Edition*)

## For Advanced Programmers

Dive deep into Ruby's metaprogramming, design patterns, and performance characteristics.

- [**Metaprogramming Ruby 2**](https://drive.google.com/file/d/1A2BgskzagF_naSqmnNRaOSyppEpogJwr/view?usp=sharing) (*2nd Edition*)
- [**Design Patterns in Ruby**](https://drive.google.com/file/d/1bO22y0x2lcdbSzsV-hss7uACKlZxE5pf/view?usp=sharing) (*1st Edition*)
- [**Refactoring: Ruby Edition**](https://drive.google.com/file/d/1h9TEOF8KUdXVqUUZs-ihB0s43MFSMtss/view?usp=sharing) (*1st Edition*)
- [**Ruby Performance Optimization**](https://drive.google.com/file/d/1QWKdMnrGSviWmEZQ_adQoVL5hlUT-D0x/view?usp=sharing) (*1st Edition*)

## For Specialized Fields

Focus on applying Ruby to a specific professional domain, especially web development with Ruby on Rails.

- [**Ruby on Rails Tutorial**](https://drive.google.com/file/d/1K6s83h9Nn9jApIMz1uuVrplTweP-rWmM/view?usp=sharing) (*7th Edition*)
- [**Agile Web Development with Rails 7**](https://drive.google.com/file/d/10Gg5TWEokpr0ZZujBmWDqUMnqJXnTlet/view?usp=sharing) (*2022 Edition*)
- [**The Rails 7 Way**](https://drive.google.com/file/d/12jcP23KAUfse3EgQR2buW78w0UhehRJT/view?usp=sharing) (*4th Edition*)
- [**Sustainable Web Development with Ruby on Rails**](https://drive.google.com/file/d/1zccKnVroVNhc0RngeczjaU58NRkIDWno/view?usp=sharing) (*1st Edition*)
- [**Effective Testing with RSpec 3**](https://drive.google.com/file/d/1b_Z0wSRlSOYzyns69uFsA7YUn3iDhBNz/view?usp=sharing) (*1st Edition*)
- [**Black Hat Ruby**](https://drive.google.com/file/d/1hN-wOW755e_qgP9oODCkoRze8G7mJuXL/view?usp=sharing) (*1st Edition*)
