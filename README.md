- **20 language guides** under [`docs/`](docs/), each with a short introduction and skill-level sections
- Paths for **Beginners**, **Intermediate**, **Advanced**, **Specialized fields**, and **References**
- Mix of **books** (mostly public Google Drive shares), **official documentation**, and other references
- Contribution guidelines that **preserve working Drive links** and reject placeholders

## How to use this repository

1. Open a language page from the [Contents](#contents) list (for example [Python](docs/python.md) or [Rust](docs/rust.md)).
2. Start with **For Beginners** if you are new; otherwise jump to Intermediate or Advanced.
3. Prefer official documentation and author-hosted free editions when both a mirror and an official source exist.
4. Found a broken link or have a better resource? Open an issue or PR — see [Contributing](CONTRIBUTING.md).

## FAQ

### Is this list free and open source?

The **repository** (this curated list and documentation) is licensed under the [MIT License](LICENSE). Individual books remain under their own copyright and publisher licenses. Always respect copyright law in your region and use legal sources.

### Are every entry free PDF downloads?

Most book entries use **public Google Drive share links** maintained for this collection. Some entries point to official docs or other references. This git repository is an index of links, not a place where PDF binaries are committed.

### Why were some links removed?

Only non-working placeholders (`PDF_LINK`, empty URLs) and unrelated dump-site hosts were removed. Working public Google Drive links are kept. Prefer opening a PR with a public Drive link or official docs URL when something is missing.

### How do I suggest a book?

Open a [book suggestion issue](.github/ISSUE_TEMPLATE/book-suggestion.yml) or submit a pull request. Follow the format and link policy in [CONTRIBUTING.md](CONTRIBUTING.md).

### How do I report a broken or malicious link?

Use the [broken link issue template](.github/ISSUE_TEMPLATE/broken-link.yml) or see [SECURITY.md](SECURITY.md) for malicious/phishing links.

### How is this different from free-programming-books?

[EbookFoundation/free-programming-books](https://github.com/EbookFoundation/free-programming-books) focuses on freely available books. This repo organizes **learning paths by language and skill level**, including recommended titles and official references. Both lists complement each other.

More answers: [docs/faq.md](docs/faq.md)
